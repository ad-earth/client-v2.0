## Development Setup

```bash

# Move to your workspace
cd your-workspace

# Clone this repository:
$ git clone https://github.com/ad-earth/client-v2.0.git

# Enter in directory:
$ cd client-v2.0

# Install dependencies:
$ yarn install

# Run the development server:
$ yarn start

# Open the project in your browser
$ http://localhost:3000
```
